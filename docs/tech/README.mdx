## 组织约定

- **本目录是事实标准**：与 `research/` 冲突时以此为准；`research/` 只作历史。
- **不重复权威内容**：本仓库产物多为 Markdown 形式的 skill/agent/prompt + JSON manifest，它们**本身就是机制的权威规约**。`tech/` 文档因此只记 skill 里未完整记录的**设计决策 / 被否方案（why）**，机制细节一律**链接**回权威源，不复述。
- **顶层 dir vs 模块 dir**：跨 skill 共享的机制（如 backlog 横跨 backlog + autonomy-controller）单独成 dir；单个组件的故事放模块 dir。
- **双端同步**：任何机制在 claude/ 与 codex/ 走同样主流程，改一侧必同步另一侧。
